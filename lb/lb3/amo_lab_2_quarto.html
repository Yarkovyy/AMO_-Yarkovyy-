# Сорутвання методом вставок (включення)


def insertion_sort(nums):
    # починаємо з другого елемент
    for j in range(1, len(nums)):

        key = nums[j]
        # Зберігаємо посилання на індекс першого елемента
        i = j - 1
        # Елементи відсортованого сегрмента переміщуємо вперед, якщо вони більше елемента вставки
        while i >= 0 and nums[i] > key:
            nums[i + 1] = nums[i]
            i -= 1
        # Вставляємо елемент
        nums[i + 1] = key

import random
from array import array
import time

def bubble_sort(array):
    length = len(array)
    for i in range(0, length):
        for j in range(0, length - i - 1):
            if array[j] > array[j + 1]:
                temp = array[j]
                array[j] = array[j + 1]
                array[j + 1] = temp

Контрольні запитання

  1. Що таке \(O-нотація\) і чим вона відрізняється від \(\Omega-нотації\)?

    \(O-нотація\) показує складність у найгіршому випадку, \(\Omega-нотація\) це складність в середньому.

  2. Яку часову складність має алгоритм сортування методом вставки за найсприятливішої умови, коли масив відсортовано? Відповідь обгрунтувати.

    Найкращий випадок: 𝑂(𝑛) – коли вихідний список вже відсортований, цикл пройдеться 1 раз по усім елементам масиву, тобто вийде n.

  3. Дати визначення ефективного алогоритму.

    Ефективний алгоритм - це алгоритм, який забезпечує оптимальні результати при мінімальному використанні ресурсів

  4. Пояснити поняття головного параметра (розміру) задачі.

    Набільш впливова частина функції часової складності.

  5. Функція часової складності має вигляд: \(F(N)=N^3+7N^2-14N\). Як записати аисмптотичну складність в нотації \(O()\)?

    Вибрати параметр який зростає найшвидше. Інші параметри, а також константи, відкинути. Складність = \(O(N^3)\).
